# RPSLS
C# project of Rock Paper Scissors Lizard Spock game

Assignment User Stories
1. find a way to properly incorporate inheritance
2. account for and handle bad user input, ensuring that any user input is validated and reobtained if necessary.
3. store all of the gesture options/choices in a List<T>.
4. correct player to win a given round based on the choices made by each player. 
5. RPSLS to be at minimum a ‘best of three’ to decide a winner.
6. option of a single player (human vs AI) or a multiplayer (human vs human) game.
